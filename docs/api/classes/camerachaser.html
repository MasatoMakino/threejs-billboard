  • lookCamera(render: WebGLRenderer, scene: Scene, camera: Camera, geometry: BufferGeometry, material: Material, group: Object3D): void
    • Defined in src/CameraChaser.ts:35

    Planeをカメラに向ける。lookCameraHorizontal = trueの時だけ稼働する。 回転方向はY軸を中心とした左右方向のみ。 (X軸方向には回転しない。X軸方向に回転させたい場合はBillBoardクラスを利用する。)

    カメラ位置がPlaneの北極、南極をまたぐと急激に回転するので注意。 利用する場合はカメラの高さ方向に制限をかけた方が良い。
