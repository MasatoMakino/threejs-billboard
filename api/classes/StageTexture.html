    An image object, typically created using the {@link TextureLoader.load} method. This can be any image (e.g., PNG, JPG, GIF, DDS) or video (e.g., MP4, OGG/OGV) type supported by three.js.

    To use video as a texture you need to have a playing HTML5 video element as a source for your texture image and continuously update this texture as long as video is playing - the {@link VideoTexture} class handles this automatically.

    Returns any
